# POS Hardware Recommendations

This document provides recommended hardware setups for merchants using the Contacto POS application. These recommendations are categorized into different budget levels to suit various business sizes and needs.

---

## **Budget Option**
-   **Total Estimated Cost:** ~50,000 - 80,000 DZD
-   **Ideal for:** Small shops, cafes, and businesses with low transaction volume.

| Component | Recommendation              | Estimated Cost (DZD) | Notes                               |
| --------- | --------------------------- | -------------------- | ----------------------------------- |
| **Tablet**  | Samsung Galaxy Tab A8       | ~50,000              | 10.5" display, Android 11+          |
| **Scanner** | Generic 1D Bluetooth Scanner | ~8,000               | Sufficient for standard barcodes.   |
| **Printer** | Generic 58mm Bluetooth Printer | ~15,000              | Basic thermal receipt printing.     |
| **Total**   |                             | **~73,000**          |                                     |

---

## **Mid-range Option**
-   **Total Estimated Cost:** ~150,000 - 200,000 DZD
-   **Ideal for:** Medium-sized retail stores, restaurants, and businesses with moderate transaction volume.

| Component     | Recommendation         | Estimated Cost (DZD) | Notes                                   |
| ------------- | ---------------------- | -------------------- | --------------------------------------- |
| **Tablet**      | Samsung Galaxy Tab S9  | ~100,000             | 11" display, Android 13+, better performance. |
| **Scanner**     | Zebra DS2278           | ~35,000              | High-performance 2D scanner.            |
| **Printer**     | Epson TM-T82III        | ~45,000              | Reliable and fast 80mm thermal printer. |
| **Cash Drawer** | Generic RJ11 Drawer    | ~20,000              | Connects to the receipt printer.        |
| **Total**       |                        | **~200,000**         |                                         |

---

## **Professional Option**
-   **Total Estimated Cost:** ~400,000+ DZD
-   **Ideal for:** High-volume retail, supermarkets, and businesses requiring a robust, all-in-one solution.

| Component        | Recommendation                           | Notes                                                        |
| ---------------- | ---------------------------------------- | ------------------------------------------------------------ |
| **POS Terminal** | All-in-one Android POS (e.g., Sunmi, PAX) | Includes a built-in printer, customer display, and integrated scanner. Provides a professional and streamlined checkout experience. |
| **Card Reader**  | Integrated EMV + Contactless (NFC)       | Securely accepts all modern forms of card payment. (Phase 3) |
| **Cash Drawer**  | Heavy-duty, high-cycle drawer            | Designed for frequent use in high-volume environments.       |
