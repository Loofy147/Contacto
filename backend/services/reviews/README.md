# Reviews Service

## Overview

This directory contains the source code for the **Reviews Service**.

The primary responsibility of this service is to manage the submission, moderation, and display of user reviews and ratings for professionals.

For a complete overview of how this service fits into the larger system, please see the main **[Technical Architecture Document](../../../docs/architecture/technical_architecture.md)**.

## Core Responsibilities

-   Allowing users to create, read, update, and delete their reviews.
-   Implementing a content moderation workflow for new reviews.
-   Calculating and updating average ratings for professionals.
-   Handling replies to reviews.
