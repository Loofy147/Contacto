# Inventory Service

## Overview

This directory contains the source code for the **Inventory Service**.

The primary responsibility of this service is to manage product and service catalogs, stock levels, and pricing for businesses.

For a complete overview of how this service fits into the larger system, please see the main **[Technical Architecture Document](../../../docs/architecture/technical_architecture.md)**.

## Core Responsibilities

-   Managing product and service information (name, description, SKU).
-   Tracking inventory levels and stock adjustments.
-   Handling price updates and different price lists.
-   Managing suppliers and purchase orders.
