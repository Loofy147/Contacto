# Employees Service

## Overview

This directory contains the source code for the **Employees Service**.

The primary responsibility of this service is to manage employee information, roles, permissions, and schedules for businesses using the platform.

For a complete overview of how this service fits into the larger system, please see the main **[Technical Architecture Document](../../../docs/architecture/technical_architecture.md)**.

## Core Responsibilities

-   Managing profiles for employees of a business.
-   Implementing Role-Based Access Control (RBAC) to define what employees can see and do.
-   Managing work schedules and shifts.
-   Tracking employee time and attendance.
