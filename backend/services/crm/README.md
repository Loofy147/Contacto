# CRM Service

## Overview

This directory contains the source code for the **CRM Service**.

The primary responsibility of this service is to manage customer relationships, including client data, interaction history, and communication.

For a complete overview of how this service fits into the larger system, please see the main **[Technical Architecture Document](../../../docs/architecture/technical_architecture.md)**.

## Core Responsibilities

-   Storing and managing customer profiles and contact information.
-   Logging all interactions with customers (e.g., appointments, messages).
-   Managing customer segments and marketing lists.
-   Providing a history of customer engagement.
