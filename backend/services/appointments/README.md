# Appointments Service

## Overview

This directory contains the source code for the **Appointments Service**.

The primary responsibility of this service is to manage the scheduling, booking, and notification of appointments between professionals and clients.

For a complete overview of how this service fits into the larger system, please see the main **[Technical Architecture Document](../../../docs/architecture/technical_architecture.md)**.

## Core Responsibilities

-   Managing the availability and schedules of professionals.
-   Handling the creation, confirmation, and cancellation of appointments.
-   Sending reminders and notifications related to appointments.
-   Checking for scheduling conflicts.
