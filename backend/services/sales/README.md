# Sales Service

## Overview

This directory contains the source code for the **Sales Service**.

The primary responsibility of this service is to manage point-of-sale (POS) operations, including processing sales, generating receipts, and managing orders.

For a complete overview of how this service fits into the larger system, please see the main **[Technical Architecture Document](../../../docs/architecture/technical_architecture.md)**.

## Core Responsibilities

-   Creating and managing sales transactions and orders.
-   Applying discounts, taxes, and tips to sales.
-   Generating and printing receipts.
-   Managing the daily sales ledger and cash drawer operations.
